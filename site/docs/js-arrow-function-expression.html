The arrow function expression (=>) creates anonymous functions who do not have their this, arguments, super, or new.target set. This is especially useful for callbacks where you don‚Äôt need all that state.

[1,2,3].forEach( num => {
  // Do something with num.
});

const someFunc = () => { /* do stuff */ };
someFunc();
